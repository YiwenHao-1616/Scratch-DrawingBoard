# 功能讲解
1.下方面板滑动条依次用于调节画笔的颜色、饱和度、亮度、粗细以及透明度；
<img width="908" height="104" alt="image" src="https://github.com/user-attachments/assets/ad44e556-6101-4374-b2b8-ed89c2345da0" />
2.右下方绿色半圆按钮用于生成作品文件，点击后会生成保存数据和"save"面板，右键可以保存/导入（格式为.txt)；鼠标移出"save"面板后面板自动关闭；
<img width="338" height="276" alt="image" src="https://github.com/user-attachments/assets/ffd950bc-0b77-49d2-a28a-761b43b8aac0" />
3.右下方橙色半圆按钮用于复现"save"面板包含的数据；
<img width="89" height="55" alt="image" src="https://github.com/user-attachments/assets/68a34537-a3de-4c03-b4af-4f9f6d728b88" />
4.橡皮擦按钮用于擦除，可拖动粗细和透明度滑动条调节橡皮擦粗细和透明度；再次点击橡皮擦按钮可以切换回画笔模式；
<img width="33" height="31" alt="image" src="https://github.com/user-attachments/assets/97615cf2-9985-4d8f-80a6-c9edf0681ef1" />
5.垃圾桶按钮用于清空屏幕；
<img width="31" height="30" alt="image" src="https://github.com/user-attachments/assets/169993aa-a1fd-40bb-b380-0fcea5f24cee" />
6.画板背景为白色。
